# 🌸 ARTAVA

**ARTAVA** is an AI-powered menstrual health companion designed to help users track, analyze, and better understand their menstrual health through intelligent insights and personalized predictions.

## Vision

To empower women with accurate, private, and data-driven menstrual health tracking.

## Planned Features

- Period Tracking
- Cycle Prediction
- Ovulation Prediction
- Flow Tracking
- Mood Tracking
- Symptom Tracking
- AI Health Insights
- Health Reports
- Notifications
- Secure Authentication

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL
- Prisma ORM

### AI
- OpenAI API (or another LLM)
- Predictive analytics
